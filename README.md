\# Centro de IA 🤖



Proyecto desarrollado en Python utilizando Machine Learning.



\## Funcionalidades



\### 🏠 Predicción de Casas

Predice el precio de una vivienda según sus metros cuadrados utilizando Regresión Lineal.



\### 📧 Detector de Spam

Clasifica mensajes como SPAM o NO SPAM mediante un modelo de clasificación.



\## Tecnologías Utilizadas



\- Python

\- Tkinter

\- Pandas

\- Scikit-Learn



\## Ejecución



Instalar dependencias:



```bash

pip install pandas scikit-learn

```



Ejecutar:



```bash

python app.py

```
